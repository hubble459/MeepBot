# MeepBot

Bot for meeping

## Commands

| music    |          | misc    | settings |
| ---      | ---      | ---     | ---      |
| play     | clear    | alias   | prefix   |
| stop     | remove   | aliasrm | space    |
| skip     | repeat   | aliases |          |
| pause    | shuffle  | echo    |          |
| resume   | playlist | meep    |          |
| queue    | volume   | ping    |          |
| now      |          |         |          |

## Install

make env for discord token

After making this file, you can install the dependencies and run the program with the following:

```shell
npm install
npm start
```
